# DJI设计系统重构 - 详细验证指南

## 🎯 关键验证点

### 1. DJI蓝色主题验证 (#0070d5)
**检查位置**：
- [ ] 主要CTA按钮 ("Add to Cart", "Shop Now"等)
- [ ] 导航链接悬停状态
- [ ] 主题切换按钮
- [ ] 链接文字颜色

**验证方法**：
1. 打开开发者工具 (F12)
2. 选择"元素"标签
3. 点击任意主要按钮
4. 在右侧样式面板检查 `background-color` 应为 `rgb(0, 112, 213)`

### 2. 胶囊型按钮验证 (1408px圆角)
**检查位置**：
- [ ] 首页产品卡片的"Add to Cart"按钮
- [ ] 产品页面的主要操作按钮
- [ ] 导航中的主要链接

**验证方法**：
1. 右键点击按钮 → "检查元素"
2. 查看 `border-radius` 值应为 `1408px`
3. 按钮应呈现完全圆润的胶囊形状

### 3. 7层阴影系统验证
**检查位置**：
- [ ] 产品卡片悬停效果
- [ ] 按钮悬停效果
- [ ] 模态框和下拉菜单

**验证方法**：
1. 悬停在产品卡片上
2. 卡片应向上移动4px (transform: translateY(-4px))
3. 阴影应变为更深的阴影效果
4. 检查CSS中的 `box-shadow` 值

### 4. 悬停动画效果验证
**检查项目**：
- [ ] 按钮悬停时向上移动2px
- [ ] 卡片悬停时向上移动4px
- [ ] 图片悬停时放大到1.05倍
- [ ] 动画时长为0.3s，缓动为ease

**验证方法**：
1. 缓慢移动鼠标到按钮上
2. 观察平滑的向上移动动画
3. 移开鼠标时应平滑恢复原位

### 5. Open Sans字体验证
**检查位置**：
- [ ] 页面标题
- [ ] 产品名称
- [ ] 按钮文字
- [ ] 导航菜单

**验证方法**：
1. 开发者工具 → "计算"标签
2. 查看 `font-family` 应包含 `'Open Sans'`
3. 字重应为400 (Regular) 或 600 (Semibold)

### 6. 8px网格间距验证
**检查位置**：
- [ ] 产品卡片内部间距
- [ ] 按钮内边距
- [ ] 导航项间距

**验证方法**：
1. 使用开发者工具测量工具
2. 元素间距应为8px的倍数：8px, 16px, 24px, 32px, 48px

## 🐛 常见问题排查

### 如果按钮不是胶囊型
**可能原因**：
- CSS类名错误
- Tailwind配置未正确加载
- 浏览器缓存问题

**解决方法**：
1. 强制刷新页面 (Ctrl+F5)
2. 清除浏览器缓存
3. 检查开发者工具中的CSS

### 如果DJI蓝色不正确
**可能原因**：
- CSS变量未正确加载
- 主题切换影响颜色
- 浏览器兼容性问题

**解决方法**：
1. 检查CSS变量值
2. 确认主题状态
3. 测试不同浏览器

### 如果动画不流畅
**可能原因**：
- 硬件加速未启用
- 浏览器性能限制
- CSS动画属性问题

**解决方法**：
1. 启用硬件加速
2. 关闭其他标签页
3. 检查动画属性
