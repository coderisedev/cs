# Medusa 商品变体 (Variant) 管理指南

本指南将详细说明如何在 Medusa Admin 后台为同一个商品创建不同的变体（例如不同颜色或尺寸）。

## 概念说明
在 Medusa 中：
- **商品 (Product)**: 抽象的产品主体 (例如: "纯棉 T恤")。
- **选项 (Options)**: 产品的属性维度 (例如: "尺码", "颜色", "材质")。
- **变体 (Variants)**: 具体的库存单元 (SKU)，即顾客实际购买的组合 (例如: "纯棉 T恤 / S码 / 红色")。

## 操作步骤

### 1. 配置商品选项 (Options)
在创建具体的变体之前，必须先定义“什么”会变化。

1.  登录 **Medusa Admin** 后台。
2.  进入 **Products** (商品) 菜单，点击您要编辑的商品。
3.  向下滚动到 **Options** (选项) 区域。
4.  点击 **Add Option** (添加选项) 或编辑现有选项。
    *   **Title (标题)**: 输入属性名称 (例如 `Size` 或 `Color`)。
    *   **Values (值)**: 输入所有可能的值 (例如 `S`, `M`, `L`, `XL`)。每输入一个值后按回车键。
5.  点击 **Save** (保存)。
    *   *重复此步骤以添加其他维度 (例如添加一个名为 "Color" 的选项，值为 "Red", "Blue")。*

### 2. 创建变体 (Create a Variant)
有了选项后，就可以创建具体的组合了。

1.  在商品详情页向下滚动到 **Variants** (变体) 区域。
2.  点击 **Create Variant** (或者 `+` 按钮)。
3.  **General Details (基本信息)**:
    *   **Title (标题)**: 通常根据选项自动生成 (例如 "S / Red")，您也可以自定义。
    *   **Options (规格)**: 为每个选项选择具体的值 (例如 Size 选 `S`, Color 选 `Red`)。
4.  **Pricing (价格)**:
    *   为支持的每种货币/地区输入价格。
    *   *注意: 不同的变体可以设置不同的价格 (例如 XL 码可以比 S 码更贵)。*
5.  **Stock & Inventory (库存)**:
    *   **SKU**: 输入唯一的库存单位编号 (例如 `TSHIRT-RED-S`)。**强烈建议填写**。
    *   **Manage Inventory (管理库存)**: 勾选此项以跟踪库存数量。
    *   **Quantity (数量)**: 如果开启库存管理，请输入当前库存。
    *   **Allow Backorders (允许缺货销售)**: 勾选后，即使库存为 0 也允许顾客购买。
6.  **Shipping (物流信息)**:
    *   **Weight/Dimensions (重量/尺寸)**: 如果您使用的是 **Dynamic Shipping Module (动态运费)** 插件，请务必填写重量。
7.  点击 **Save** (保存)。

### 3. 管理现有变体
在 **Variants** 列表可以查看所有已创建的变体。

*   **编辑**: 点击某一行变体，可以修改它的价格、库存或元数据。
*   **删除**: 选中不再销售的变体并删除。
*   **复制**: (如果版本支持) 可以快速复制一个相似的变体，只需修改 SKU 和某个选项值。

## 最佳实践建议
1.  **规范 SKU 命名**: 使用类似 `分类-商品-颜色-尺码` 的格式 (例如 `TS-001-BLK-M`)。这有助于仓库管理和发货。
2.  **绑定图片**: 建议在变体详情中绑定对应的图片，这样当用户选择 "红色" 时，前台会自动切换显示红色衣服的图片。
3.  **完整设置价格**: 确保为所有支持的货币都设置了价格，否则国际客户在结账时可能会报错。

## 常见问题排查
*   **"我无法选择 'Red' 红色?"**: 请回到 **Options** 区域，确保 "Color" 选项下已经添加了 "Red" 这个值。
*   **"运费计算不正确?"**: 请检查该变体是否填写了 **Weight (重量)**，我们的自动运费插件依赖此数据进行计算。
